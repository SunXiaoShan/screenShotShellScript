# screenShotShellScript

## Getting Started
Step 1: Chmod 

Step 2: Add new URLs you want to get the screen shots on screenShot.sh
```
# URL list
URLS="https://www.google.com/ \
https://tw.yahoo.com/ \
"
```
Step 3: run it
